# Random generators study We use different random generators to get coords of dots that we plot in 256x256 squares. ## Linear congruential method Starting with an arbitrary seed X_0, we compute the next random vallue with this formula: X_{n+1} = (a X_n + c) \% m Let __m__ be 2^32. Our random generator will have a period of __m__ if and only if: * __c__ is relatively prime to __m__. _(__c__ must be odd.)_ * __a - 1__ is a multiple of every prime dividing __m__. _(__a__ must be odd)_ * If __m__ is a multiple of 4, __a - 1__ is a multiple of 4. _(__a = 4.k + 1__)_ ## Mitchell and Moore method For any n > 54, we apply this formula: X_n = (X_{n-24} + X_{n-55}) \% m The seed is made of the 55 first element of X_n. The only constraint is that there must be at least one odd number. Otherwise, all the generated numbers will be even.
